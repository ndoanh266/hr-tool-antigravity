Quy Tắc: Chấm Điểm CV Theo Tiêu Chí

Quy Trình Thực Hiện (Tối Ưu Chi Phí):

  1. Bước 1: Trích xuất nội dung từ file CV
    Chạy: python scripts/extract_text.py --input "<đường_dẫn_cv>"
    Mục đích: Lấy văn bản thô của file CV.
  2. Bước 2: AI Agent (Bạn) trực tiếp chấm điểm CV
    Mục đích: Bạn tự so sánh nội dung CV vừa đọc được với các tiêu chí do người dùng đưa ra, thực hiện đánh giá, cho điểm số chi tiết từng tiêu chí (dạng điểm/tổng điểm) và viết nhận xét. Không dùng script API bên ngoài.
