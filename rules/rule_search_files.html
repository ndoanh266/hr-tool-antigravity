Quy Tắc: Tìm Kiếm File Theo Nội Dung

Quy Trình Thực Hiện:

  1. Bước 1: Xác định thư mục quét và từ khóa cần tìm
    Nhận thông tin thư mục nguồn (quét đệ quy) và từ khóa/cụm từ cần tìm từ người dùng.
  2. Bước 2: Thực thi script tìm kiếm file
    Chạy: python scripts/search_files.py --dir "<đường_dẫn_thư_mục>" --keyword ""
    Mục đích: Quét đệ quy toàn bộ thư mục để tìm các file chứa từ khóa (hỗ trợ .pdf, .docx, .doc, .xlsx, .xls, .csv, .txt).
  3. Bước 3: Tổng hợp và hiển thị kết quả
    Báo cáo danh sách tất cả các đường dẫn file chứa từ khóa tìm được cho người dùng.
