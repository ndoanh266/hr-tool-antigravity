Quy Tắc: Trích Xuất Thông Tin Ứng Viên Từ CV Vào Excel/CSV

Quy Trình Thực Hiện (Tối Ưu Chi Phí):

  1. Bước 1: Lấy các trường dữ liệu (tiêu đề cột) từ file Excel đích
    Chạy: python scripts/get_excel_fields.py --excel "<đường_dẫn_excel>"
    Mục đích: Lấy danh sách cột hiện tại để tránh làm lệch cột khi ghi dữ liệu.
  2. Bước 2: Trích xuất nội dung văn bản từ file CV
    Chạy: python scripts/extract_text.py --input "<đường_dẫn_cv>"
    Mục đích: Đọc nội dung thô từ file PDF, DOCX, DOC hoặc XLSX/CSV.
  3. Bước 3: Dịch thuật và phân tích thông tin CV trực tiếp bằng trí tuệ của AI Agent (Bạn)
    Mục đích: AI Agent tự động dịch CV sang tiếng Việt và trích xuất thông tin khớp chính xác với tiêu đề cột thu được ở Bước 1 và các quy tắc định dạng bên dưới. Không sử dụng script API bên ngoài để tiết kiệm chi phí.
    Đầu ra: Một bảng Markdown gồm dòng tiêu đề cột và duy nhất 1 dòng dữ liệu của ứng viên.
  4. Bước 4: Lưu dữ liệu đã phân tích vào file Excel đích
    Chạy: python scripts/write_to_excel.py --excel "<đường_dẫn_excel>" --headers "" --data ""
    Mục đích: Ghi dữ liệu vào file Excel đích an toàn.
